# 📊 Daily Business Ledger Pro with Gen-AI

A sleek, premium full-stack web application designed for small commercial shops to track daily transactions and receive automated financial health audits powered by AI.

## 🚀 Visual Preview
---

## 🛠️ Tech Stack & Architecture
- **Backend Framework:** FastAPI (Python)
- **AI Inference Engine:** Groq API (`llama-3.3-70b-versatile` / `llama-3.1-8b-instant`)
- **Frontend Dashboard:** Modern Semantic HTML5 & CSS3 (Custom Dark Space Theme)
- **Environment Management:** Isolated Python `.venv` configuration

## 🌟 Key Features
- **Live Metric Counter:** Auto-calculates total cash inflow, outflow, and absolute net margins in real-time.
- **Client-Side Filter Engine:** Instant keyword parsing to browse ledger history logs without server delays.
- **Timestamp Logging Architecture:** Automates standardized transaction formatting with local browser clock tracking.
- **One-Click AI Consulting Audit:** Communicates instantly via secure endpoints to output single-sentence business summaries.
